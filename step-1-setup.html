Step 1 — Setup

A Default-style agent that helps citizens with council tax. We'll wire in knowledge, a workflow and a search tool in the next steps.

Step 1.1 — Create the agent

Open the Build area first
From any page in watsonx Orchestrate, click the hamburger menu (☰) in the top-left and choose Build — that’s where you create and manage agents.
  1. From the hamburger menu on the left, choose Agents.
  2. Click Create Agent → Create from scratch.
  3. Fill in the Name and Description fields below.
Agent name[Your initials] Council Tax Assistant
