Lab overview

In this lab you'll build a single agent in the no-code Agent Builder. It takes a transcript or set of rough meeting notes and turns them into a clean summary with decisions, action items and follow-ups — the kind of write-up you'd actually send round after a meeting.

This is your first pass through the Agent Builder. We'll keep it deliberately simple: one Default-style agent, no tools, no collaborators. Lab 2 layers in multi-agent collaboration and external tools.

What you'll learn

Agent basics
Name, description, style — what each field does and why it matters.
Behaviour
The instructions that define how your agent thinks and responds.
Guidelines
If-this-then-that rules that handle edge cases consistently.
Testing
The built-in playground for trying real prompts.

Architecture — one agent

Agentmeeting-summariser — Default style, no tools, no collaborators
InputRough meeting notes pasted by the user (a transcript, bullet points, or rambling notes)
OutputA clean summary with sections for Summary, Decisions, Action items, and Follow-ups
